What does Sole offer?

Sole is a messaging app where you own all the data. The only thing you need is a solid POD.
Solid (derived from "social linked data") is a proposed set of conventions and tools for building decentralized social applications based on Linked Data principles. Solid is modular and extensible and it relies as much as possible on existing W3C standards and protocols.

At a glance, here is what makes Sole different...

No limitations

Your data, your rules. Sole Project uses Solid standards for creating the chats, this way you can choose any client you want for interacting with it. You can even create a Sole client yourself!

True data ownership

Thanks to the Solid platform every user owns their own information and has full control over it. Sole automatically sets permissions on the data it generates to ensure only the people involved can access it.

Intuitive

The developers of Sole have put a lot of thought and effort into making Sole easy to use for everyone. Sole has a sleek aesthetic design and following preconcieved notions of what a chat application is supposed to look like.
